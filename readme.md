# 크롤링 연습

## 사용하고 있는 라이브러리 📚

- cheerio
- axios

## 작업 Flow 🧑🏼‍💻

- axios를 사용해서 해당 url 페이지에 요청을 보낸다.
- cheerio를 통해 html의 데이터에 접근한다.
- tag와 class로 가져오려는 html DOM 요소의 contents들을 탐색한다.

## 추후 작업

- [ ] 다양한 페이지들을 크롤링해보자.
- [ ] 페이지 네이션이 있는 사이트도 크롤링.
- [ ] 크롤링 한 데이터들을 .json 파일에 저장.
- [ ] 크롤링 한 데이터들을 DB에 밀어넣기.
